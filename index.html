Автоматизация — построенная регионом, для региона.

Qadam Flow — open-source платформа для автоматизации рабочих процессов. Соединяет приложения и сервисы визуальными потоками. Локализованная, self-hosted, без платных тарифов и enterprise-замков.

Зачем это нужно

Регион запускает свою автоматизацию на чужой инфраструктуре.

Биллинг в долларах, серверы за рубежом, языки не наши, а функции которые действительно важны — SSO, RBAC, audit logs — заперты за enterprise paywall.

Qadam Flow — это собственная платформа региона: локализованная, self-hosted, данные остаются дома, замки сняты.

Дело не в «бесплатном Zapier» — дело в независимости.
01
Данные остаются у вас. Self-hosted развёртывание — на вашей инфраструктуре, в вашей юрисдикции.
02
Никаких enterprise-замков. SSO, RBAC, audit logs — в roadmap, никогда за paywall.
03
Языки региона. Интерфейс на русском, узбекском, казахском, английском.
04
Региональные интеграции. Payme, Click, Uzum, Kaspi и локальные API — приоритет, а не «когда-нибудь».

Быстрый старт

Один Docker-compose и вы внутри.

Нужен Docker. Клонируйте репозиторий, скопируйте конфиг, поднимите — приложение доступно по адресу http://localhost:8080.

Полная документация
$ git clone https://github.com/aiqadam/qadam-flow.git $ cd qadam-flow $ cp .env.example .env # адаптируйте под себя $ docker compose up -d # Откройте http://localhost:8080

Что готово, что в работе

MIT-ядро Activepieces форкнуто и очищено от enterprise-кода. Локализация и региональные интеграции — текущий фокус.

Enterprise-функции (SSO / RBAC / аудит) собираются по мере реальных потребностей развёртываний — не по фиксированному графику. Подробности в Roadmap.

Поддерживается AI Qadam — пока под опекой основателя, но устроено так, чтобы принадлежать сообществу (DCO, без передачи копирайта).

Как присоединиться

Не обязательно быть экспертом — достаточно того, что вам не всё равно.

Простой и весомый старт:

Пишите Pieces

Соберите интеграцию которая нужна региону — Payme, Click, Uzum, Kaspi, локальные API.

Переводите интерфейс

Локализация — без кода. Узбекский, казахский, русский, английский.

Тестируйте и репортите

Запустите, сломайте, оформите issue. Воспроизводимый баг — половина исправления.
